Admin Panel

When creating LezFindr, we saw a need for an admin panel to monitor and approve (or deny) any new users or user reports that came in. Originally I had hoped to find an out of the box soultion but there was none, so I went ahead and designed my own. As an easy addition to the mobile app interface and secured to only specific account access, the admin panel was quickly added into the app as well as a web app for easy access.
